7、删除重复的电子邮箱

delete语句中加from和不加from的区别：

1、假如只针对一个表进行删除加不加from都一样

2、如果需要联合其他表，则需要使用from，具体如本题

解法1:自连接

delete 
    p1
from
    Person p1,
    Person p2
where
    p1.email = p2.email and
    p1.id > p2.id
    ;

解法2:使用group by 进行去重操作

delete from
    Person
where 
    id not in (
        select id from (
            select min(id) id  from Person group by Email 
        ) t
    )
;

8、修复表中的名字

解法1:使用concat+upper+lower函数组合

concat:拼接字符串

upper：字母转大写

lower：字母转小写

left:从左边截取字符

right:从右边截取字符

length：获取字符串长度

# Write your MySQL query statement below
select
    user_id,
    concat(upper(left(name,1)),lower(right(name,length(name) - 1))) name
from
    users
order by
    user_id
    ;

9、按日期分组销售产品

解法1:使用group_concat函数

group_concat:将组中的字符串连接成为具有各种选项的单个字符串

order by 中如果列名是字符串类型，则按照字典序排列

separator:常与group_concat函数联合使用表示将每一个字符串中间用什么符号隔开(不加的话默认是逗号)

select
    sell_date,
    count(distinct product) num_sold,
    group_concat(distinct product order by product separator ',') products
from
    Activities
group by
    sell_date
order by
    sell_date    
    ;

10、患某种疾病的患者

解法1:使用like

select
    *
from
    Patients
where
    conditions like 'DIAB1%' or conditions like '% DiAB1%'
    ;

解法2: locate函数

locate(substr,str):返回strbstr在字符串str中第一次出现的位置，如果没有出现则返回0 

select
    *
from
    Patients
where
    left(conditions,5) = "DIAB1" or locate(" DIAB1",conditions) != 0
    ;

解法3：正则表达式

^表示以什么开头

|：或者

\s:空格

下面的正则意为：以DIAB1开头或者包含“ DIAB1”的字符（注意有空格）

select
    *
from
    Patients
where
    conditions regexp '^DIAB1|\\sDIAB1'
    ;

     select employee_id from Employees

    ;

12、树节点

解法1:case语句

# Write your MySQL query statement below
select
    id,
    case
        when t.p_id is null
            then 'Root'
        when t.p_id is not null and t.id in (select at.p_id from tree at)
            then 'Inner'
        else 'Leaf'
    end as Type
from
    tree t
order by
    t.id
    ;

解法2:if函数

select
    t.id,
    IF(isNull(t.p_id),'Root',IF(t.id in (select p_id from tree),'Inner','Leaf')) Type
from
    tree t
order by 
    t.id
    ;

13、第二高的薪水

            Salary desc
        limit 1,1
    ),null) SecondHighestSalary

解法2:去除掉最大的，然后找最大的，使用max函数

select
   max(salary) SecondHighestSalary
from
    Employee
where 
    salary != (select max(salary) from Employee)
    ;

17、上升的温度

解法1:datediff函数

datediff(d1,d2):计算出日期d1与日期d2的差值，如果d1大于d2结果为正，否则为负

select 
    w1.id
from
    Weather w1
where
    w1.Temperature > (select w2.Temperature from Weather w2 where datediff(w1.recordDate,w2.recordDate) = 1)
    ;

19、查询近30天活跃用户数

解法1:datediff()函数

select
    activity_date day,
    count(distinct user_id) active_users
from
    Activity 
where
    datediff('2019-07-27',activity_date) < 30 and datediff(activity_date,'2019-07-27') < 0
group by
    activity_date
    ;

解法2:between关键字

mysql中的between前后都是闭区间

# Write your MySQL query statement below

select
    activity_date day,
    count(distinct user_id) active_users
from
    Activity 
where
    activity_date between '2019-06-28' and '2019-07-27'
group by
    activity_date
    ;

24、2020年最后一次登录

select 
    user_id,max(time_stamp) last_stamp
from
    Logins
where
    time_stamp between '2020-01-01 00:00:00' and '2020-12-31 23:59:59'

    user_id
